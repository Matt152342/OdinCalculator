# OdinCalculator
Simple Calculator
A lightweight, browser-based calculator built to practice State Management and DOM Manipulation.

## Technologies Used
- HTML5

- CSS3

- JavaScript

## Core Logic
The application operates as a State Machine using three primary variables: number1, number2, and operand. Event listeners capture button clicks. Ensures only " .includes('.') ". Prevents multiple operands from being selected. Uses an operate() function that processes the math based on the selected operand. The UI updates dynamically to show the current entry and the overall calculation progress.
